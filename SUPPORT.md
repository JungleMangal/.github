# Getting help

## I have a question about how to use a tool

- Check the repo's `README.md` first.
- Check existing issues — someone may have asked already.
- For app-specific questions, open a Q&A discussion (or an issue with
  the `question` label) on the relevant repo.

## I think I found a bug

Open an issue on the affected repo using the **Bug report** template.
Include reproduction steps, expected vs. actual behavior, and version
info.

## I want a feature

Open an issue using the **Feature request** template. Describe the
problem you're trying to solve, not just the solution you have in
mind.

## I think I found a security issue

**Do not open a public issue.** See
[SECURITY.md](./SECURITY.md).
